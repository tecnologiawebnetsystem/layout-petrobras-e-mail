# Python Scripts

Esta pasta contém scripts Python para processamento de dados, análises e automações do sistema.

## Estrutura

- `/data-processing` - Scripts de processamento de dados
- `/migrations` - Scripts de migração de banco de dados
- `/analytics` - Scripts de análise e relatórios
- `/automation` - Scripts de automação e tarefas agendadas

## Requisitos

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
pip install -r requirements.txt
```

## Uso

Cada script deve ter documentação própria explicando sua funcionalidade e como executá-lo.
